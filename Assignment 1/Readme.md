# HTML Text Formatting Tags

## Project Title
HTML Text Formatting Tags in Tabular Format

## Description
This project demonstrates various HTML text formatting tags using a well-organized table. Each row explains a formatting tag, its purpose, the HTML code used, and the rendered output. The webpage is styled using CSS to improve readability and presentation.

## Objective
- Learn the commonly used HTML text formatting tags.
- Understand the purpose of each formatting tag.
- Display HTML tags, descriptions, code examples, and outputs in a tabular format.
- Practice basic HTML and CSS styling.

## Technologies Used
- HTML5
- CSS3

## HTML Formatting Tags Included
- `<b>` – Bold text
- `<strong>` – Important text
- `<i>` – Italic text
- `<em>` – Emphasized text
- `<mark>` – Highlighted text
- `<small>` – Smaller text
- `<del>` – Deleted text
- `<ins>` – Inserted text
- `<sub>` – Subscript text
- `<sup>` – Superscript text
- `<u>` – Underlined text
- `<code>` – Computer code text

## Features
- Simple and responsive table layout.
- Clean CSS styling.
- Displays:
  - HTML Tag
  - Description
  - Example Code
  - Rendered Output
- Easy to understand for beginners.

## How to Run
1. Download or clone the project.
2. Open `index.html` in any modern web browser.
3. The webpage will display a table containing HTML formatting tags with examples.

## Output
The webpage displays a formatted table showing:
- HTML formatting tag
- Description
- Example HTML code
- Actual rendered output

## Learning Outcomes
- Understand the difference between semantic and non-semantic formatting tags.
- Learn how HTML formatting tags affect webpage content.
- Gain experience with HTML tables and CSS styling.
