# Repository Description

This repository contains three Full Stack Development Lab (FSDL) assignments demonstrating the fundamentals of **HTML, CSS, JavaScript, Bootstrap, and Chart.js**. Each assignment focuses on different web development concepts, including HTML formatting, data visualization, and responsive form design.

---

# Assignments Included

## Assignment 1: HTML Text Formatting Tags

### Description
This assignment demonstrates various HTML text formatting tags using a structured table. Each formatting tag is explained with its description, example code, and rendered output.

### Technologies Used
- HTML5
- CSS3

### Features
- HTML formatting tags in tabular format
- Example code for each tag
- Rendered output
- Clean table styling using CSS

### Topics Covered
- `<b>`
- `<strong>`
- `<i>`
- `<em>`
- `<mark>`
- `<small>`
- `<del>`
- `<ins>`
- `<sub>`
- `<sup>`
- `<u>`
- `<code>`

---

## Assignment 2: Interactive Weather Dashboard

### Description
This project is an interactive weather dashboard that visualizes weekly weather data for different cities using **Chart.js**. Users can select a city from the dropdown menu, and the charts update dynamically.

### Technologies Used
- HTML5
- CSS3
- JavaScript
- Chart.js

### Features
- City selection (Pune, Mumbai, Delhi)
- Temperature Line Chart
- Humidity Bar Chart
- Rainfall Bar Chart
- Dynamic chart updates
- Responsive dashboard layout

### Weather Parameters
- Temperature (°C)
- Humidity (%)
- Rainfall (mm)

---

## Assignment 3: Student Registration Form using Bootstrap

### Description
This assignment demonstrates how to create a responsive student registration form using **Bootstrap 5**. It includes commonly used form components and Bootstrap styling classes.

### Technologies Used
- HTML5
- Bootstrap 5

### Features
- Responsive registration form
- Name, Email, and Password fields
- Gender selection using radio buttons
- Course selection dropdown
- Submit and Reset buttons
- Bootstrap form styling

### Bootstrap Components Used
- Container
- Form Control
- Form Label
- Form Select
- Form Check
- Buttons
- Grid and spacing utilities
