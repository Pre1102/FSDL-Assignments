# Student Registration Form using Bootstrap

## Project Title
Student Registration Form using HTML and Bootstrap 5

## Description
This project is a simple and responsive **Student Registration Form** built using **HTML5** and **Bootstrap 5**. It demonstrates the use of Bootstrap form components such as text fields, email input, password field, radio buttons, dropdown menus, and styled buttons to create a clean and user-friendly registration form.

## Objective
- Design a responsive student registration form.
- Learn how to use Bootstrap form classes.
- Understand different Bootstrap input components.
- Practice creating professional-looking web forms.

## Technologies Used
- HTML5
- Bootstrap 5 (CDN)

## Features
- Responsive layout using Bootstrap.
- Name, Email, and Password input fields.
- Gender selection using radio buttons.
- Course selection using a dropdown menu.
- Submit and Reset buttons with Bootstrap styling.
- Simple and clean user interface.

## Form Fields
- **Name** – Text input
- **Email** – Email input
- **Password** – Password input
- **Gender** – Male/Female radio buttons
- **Course** – Dropdown selection
- **Submit Button** – Submits the form
- **Reset Button** – Clears all input fields

## Bootstrap Components Used
- `container`
- `form-control`
- `form-label`
- `form-select`
- `form-check`
- `form-check-inline`
- `btn`
- `btn-primary`
- `btn-secondary`
- `mb-3`
- `text-primary`

## How to Run
1. Download or clone the project.
2. Ensure you have an internet connection (required to load the Bootstrap CDN).
3. Open `index.html` in any modern web browser.
4. Fill in the form fields and test the Submit and Reset buttons.
