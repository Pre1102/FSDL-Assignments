# Interactive Weather Dashboard

## Project Title
Interactive Weather Dashboard using HTML, CSS, JavaScript, and Chart.js

## Description
The **Interactive Weather Dashboard** is a web application that visualizes weather data for different cities using interactive charts. Users can select a city from a dropdown menu, and the dashboard dynamically updates the temperature, humidity, and rainfall charts without refreshing the page.

## Objective
- Display weather information in an interactive dashboard.
- Visualize weather data using charts.
- Allow users to switch between different cities.
- Practice JavaScript DOM manipulation and Chart.js integration.

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js

## Features
- Interactive city selection (Pune, Mumbai, Delhi).
- Dynamic chart updates without page reload.
- Three weather parameters displayed:
  - 🌡️ Temperature (Line Chart)
  - 💧 Humidity (Bar Chart)
  - 🌧️ Rainfall (Bar Chart)
- Responsive dashboard layout using CSS Grid.
- Clean and user-friendly interface.

## Weather Parameters
- **Temperature (°C):** Displays daily temperature for one week.
- **Humidity (%):** Shows daily humidity levels.
- **Rainfall (mm):** Displays daily rainfall data.

## How to Run
1. Download or clone the project.
2. Ensure you have an internet connection (required to load the Chart.js CDN).
3. Open `index.html` in any modern web browser.
4. Select a city from the dropdown menu to view its weather data.

## Dashboard Components
- **City Selector:** Choose between Pune, Mumbai, and Delhi.
- **Temperature Chart:** Line chart showing weekly temperatures.
- **Humidity Chart:** Bar chart showing weekly humidity.
- **Rainfall Chart:** Bar chart showing weekly rainfall.

## Sample Cities
- Pune
- Mumbai
- Delhi

- Learn dynamic data visualization using JavaScript.
- Practice updating charts based on user input.
